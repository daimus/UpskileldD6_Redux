# Event Handling & React Hooks

## Instalasi
1. Clone repository
```
git clone https://github.com/daimus/UpskilledD4_EventHooks
```
2. Install dependency
```
npm install
atau
yarn install
```
3. Jalankan aplikasi
```
npm run start
atau
yarn start
```
